# Jiayun 的面团手记

一个静态内容网站原型，面向吐司、披萨、软欧与欧式面包的配方、烘焙知识和制作记录。

## 本地预览

```bash
python3 -m http.server 4173
```

访问 `http://localhost:4173`。

## 部署

本项目不需要安装依赖或执行构建命令，可直接导入 Vercel。
